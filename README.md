# Azure Virtual Desktop End-to-End Deployment

## Project Overview
This project demonstrates a complete Azure Virtual Desktop (AVD) deployment in Microsoft Azure using Active Directory Domain Services, Microsoft Entra Connect Sync, FSLogix Profile Containers, and Azure Virtual Desktop Workspace.

## Technologies Used
- Microsoft Azure
- Azure Virtual Desktop (AVD)
- Active Directory Domain Services (AD DS)
- Microsoft Entra ID
- Microsoft Entra Connect Sync
- FSLogix Profile Containers
- Azure Files
- Azure Virtual Network
- Windows Server 2022
- Windows 11 Enterprise Multi-session

## Project Features
- Created Azure Resource Group
- Configured Virtual Network and Subnet
- Deployed Domain Controller VM
- Installed and Configured Active Directory
- Configured Microsoft Entra Connect Sync
- Created Azure Virtual Desktop Host Pool
- Deployed Session Host VM
- Joined Session Host to Active Directory Domain
- Configured Azure Files
- Configured FSLogix Profile Containers
- Created Application Group and Workspace
- Assigned Users
- Successfully logged into Azure Virtual Desktop through Remote Desktop Client
